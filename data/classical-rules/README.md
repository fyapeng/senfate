- 图书：7；
- 语义片段：15,967；
- 来源记录：37,231；

## 文件

- `classical-source-corpus.v4.0.json.gz`：来源记录、枚举、条件字典、术语字典和家族索引；
- `manifest.json`：编码、规模、字节数和校验值；
- `enums.json`：人工查看用枚举表。

每条来源记录进入统一参考函数管线，并被判定为运行候选、家族证据、待补全、待裁决或技术无效。重复来源保留 provenance，但不产生重复模型作用。

## 边界
