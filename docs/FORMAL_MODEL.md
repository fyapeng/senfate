# SenFate 形式模型

## 1. 对象层与经验层分离

SenFate 先研究传统八字理论内部究竟定义了哪些对象、条件和推断，再讨论这些输出与现实观测之间是否存在统计关系。形式化提高可读性、一致性和可复算性，不构成经验有效性的证明。

```text
传统理论对象层                         独立经验研究层
出生时空 → 状态演化 → 主题泛函        可观察事件 → 估计量 → 校准与检验
```

对象层输出可以成为经验模型中的解释变量、处理变量或分层变量。任何概率、因果效应或预测性能都必须由第二层的数据和识别设计产生。

## 2. 状态空间

阶段状态写为：

```text
Z_t = (G_t, μ_t, R_t, M, P_t)
```

- `G_t`：带属性的干支—藏干关系图；
- `μ_t`：有限五行空间上的正测度或有符号测度；
- `R_t`：经过竞争、阻断和成化裁决的关系集合；
- `M`：公开且版本化的模型参数；
- `P_t`：规则来源、阶段和计算证书。

五行、主题等集合是有限可测空间。所谓“权重”只有在指定模型 `M` 后才有意义，不作为自然常数。

## 3. 非自治演化

原局、大运、流年采用依赖当前阶段的算子：

```text
Z_N       = T_N(X, M)
Z_(N,D)   = T_D(Z_N, D, M)
Z_(N,D,Y) = T_Y(Z_(N,D), Y, M)
```

这是一条路径依赖的非自治演化链。年度状态包含原局与当前大运留下的结构，不能改写为三个互相独立分数的相加。

## 4. 正规形与失败关闭

结构算子族在有限状态指纹上迭代，直到：

- `stable`：发布稳定正规形；
- `cycle`：状态指纹重复，失败关闭；
- `limit-reached`：达到迭代上限，失败关闭。

主题层的定义域只包含 `stable`。原始边、未裁决候选和失败状态不属于主题泛函的定义域。

## 5. 主题泛函与贡献测度

对主题 `k` 定义版本化泛函：

```text
Φ_(k,M) : ResolvedFeatureSnapshot → TopicProposition_k
```

规则家族求值形成有限主题空间上的有符号贡献测度：

```text
ν_t(A) = Σ_f s(f, t, M) · 1[domain(f) ∈ A]
```

其中 `s` 只表示传统模型内部的支持、压力或零作用。它不是事件概率。多本书对同一家族的重复表述增加来源覆盖，不重复增加 `ν_t`。

## 6. 时间不确定性

出生钟表时间、坐标和历表误差表示为闭区间。若区间穿过节气、换日或时辰边界，Calendar Engine 返回 `boundary-ambiguous`，要求用户提高精度或比较多个合法状态。

## 7. 跨学科接口

完成对象层后，可以建立独立研究数据合同：

- 统计学：检验主题泛函与预注册事件的关联和校准；
- 计量经济学：明确处理、结果、混杂与识别假设，避免事后解释；
- 地理学：研究地点、时区制度和环境变量，而不把经纬度修正误当作因果机制；
- 生物与健康研究：只使用合规、知情同意和可撤回的数据，不把命理主题作为医学诊断。

这些学科可以分析“传统体系编码了什么”和“输出是否与观察相关”，不能由形式符号直接推出经验真实性。
